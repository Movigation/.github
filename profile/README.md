## 프로젝트 소개

무비서(MovieSir)는 사용자의 이동 시간에 맞춰 최적의 콘텐츠를 추천해주는 OTT 서비스입니다

AI 기반 추천 알고리즘(LightGCN, Sentence-BERT)을 활용하여 개인화된 영화 추천을 제공합니다
